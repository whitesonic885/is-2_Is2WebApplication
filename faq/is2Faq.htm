iSTAR-2パッケージダウンロードについて
Ｑ
どこからダウンロードをすればいいですか？
Ａ

iS-2ユーザ専用ページの左側メニュー欄[ダウンロード]バナーをクリックし、[iS-2ダウンロードページ]画面からパッケージのダウンロードを行ってください。
Ｑ
どのファイルをダウンロードすればいいですか？
Ａ iS-2パッケージ、.NET Framework 1.1（インストールされていない場合）、ご使用になるラベルプリンタドライバをダウンロードしてください。（iS-2は.NETで開発しております。お使いになるパソコンに.NET Frameork 1.1がインストールされていない場合、iS-2のインストール前に.NET Framework 1.1のインストールを行ってください）

iSTAR-2インストールについて
Ｑ
どのようにインストールを行うのですか？
Ａ

インストールするには管理者権限が必要です。インストールの詳細についてはiS-2ユーザ専用ページの左側メニュー欄[マニュアル]バナーをクリックし[iSTAR-2セットアップ手引きダウンロード]を参照してください。

Ｑ
iS-2のインストール途中、「このセットアップは、.NET Frameworkバージョン 1.1を必要とします。.NET Frameworkをインストールして、このセットアップをやり直してください。.NET FrameworkはWebから取得できます。今すぐ取得しますか？」と表示されたが、どうすればいいですか？
Ａ メッセージ画面を[いいえ]で閉じ、iS-2ユーザ専用ページの左側メニュー欄[ダウンロード]バナーをクリックし[iS-2ダウンロードページ]画面から、.NET Framework 1.1のパッケージをダウンロード後、iS-2のインストール前に.NET Framework 1.1のインストールを行ってください。

Ｑ
.NET Framework 1.1のインストール途中、[使用許諾契約書]が表示されませんが、どうすればいいですか？
Ａ すでに、お客様のパソコンに.NET Framework 1.1がインストールされている場合は表示されません。
.NET Framework 1.1のインストールは必要ありませんので、iS-2もしくはサーマルプリンタのインストールを行ってください。

プリンタインストールについて

Ｑ
どのようにインストールを行うのですか？
Ａ

iS-2ユーザ専用ページからダウンロードしたプリンタドライバの保存先を開き、（SATO製プリンタの場合：表面が青）[printsat.exe]を、ダブルクリックすると、プリンタドライバの解凍が行われ「Lespritβ_V90050」フォルダが解凍されます。（Zebra製の場合：表面が白）[printdev.exe]をダブルクリックすると、プリンタドライバの解凍が行われ「ゼブラユニバーサルドライバ」フォルダに解凍されます。解凍後、プリンタのインストールを行ってください。

Ｑ
プリンタドライバの解凍途中、「発行元が確認できませんでした。このソフトウェアを実行しますか？」と表示されたが、どうすればいいですか？
Ａ Windowsの警告画面ですので[続行]をクリックし、インストールを続けてください。

Ｑ
プリンタのインストール途中、「このハードウェア：プリンタを使用するためにインストールしようとしているソフトウェアは、…Windowsロゴテストに合格していません。」と表示されたが、どうすればいいですか？
Ａ Windowsの警告画面ですので[続行]をクリックし、インストールを続けてください。

Ｑ
プリンタのインストール途中、「デジタル署名が見つかりませんでした…インストールを続行しますか？」と表示されたが、どうすればいいですか？
Ａ Windowsの警告画面ですので[続行]をクリックし、インストールを続けてください。

Ｑ
プリンタのインストール途中、「スプーラに印刷ジョブが残っています」と表示されたが、どうすればいいですか？
Ａ パソコンと接続しているプリンタに、印刷されていないデータが残っていますので全て印刷するか、印刷ドキュメントを削除してからインストールを行ってください。

iSTAR-2起動について

Ｑ
どのように起動すればいいですか？
Ａ ｉS-2インストール後、赤色と緑色の[iS-2]アイコンがデスクトップに表示されます。通常は赤色のアイコンをダブルクリックしiS-2の起動となります。ログイン画面が表示されましたら、ユーザーコード・パスワードを入力しログインを行ってください。緑色のアイコンは[iS-2（クイックエントリー）]となります。

Ｑ
iS-2を起動させようとデスクトップの[iS-2]アイコンをダブルクリックしたら、「サーバーとの通信に失敗しました。ＬＡＮケーブルやネットワーク設定等を確認してください」と表示されたが、どうすればいいですか？
Ａ

原因が複数あるため、下記の項目をご確認ください。

  • Windowsインターネットエクスプローラーからの検索は、正常にできていますか？iS-2はインターネットに接続できなければ使用できません。
  • タスクバーの右端に表示されているシステム日付は、現在の正確な日付をさしていますか？
  • iS-2の二重起動の確認はされましたか？
  • 自動構成スクリプトを使用していませんか？（iS-2は対応しておりません）
  • プロキシサーバーの設定を行っていませんか？（プロキシ設定を行っている場合、ポート443(https)、iS-2サーバー(wwwis2.fukutsu.co.jp)と通信が行えるように設定してください）
  • お使いのセキュリティーソフトのファイアーウォール機能の設定で、iS-2の通信が遮断されていませんか？（遮断されている場合は、許可設定を行ってください）
  • iS-2推奨環境は整っていますか？（OS・CPU・メモリ・通信回線など、動作環境はiS-2ホームページに記載していますので、ご確認ください）
  • iS-2以外のプログラムが起動していませんか？（他のプログラムと同時起動していると、iS-2が起動できない場合があります。他のプログラムは終了させてください）
  • OSがWindows Vista,Windows 7の場合、ユーザーアカウント制御（UAC）は無効化にしてiS-2をインストールされましたか？（iS-2はユーザーアカウント制御（UAC）が有効化の場合、一時的にユーザーアカウント制御（UAC）を無効化にしてインストールする必要があります）インストールの詳細についてはiS-2ユーザ専用ページの左側メニュー欄[マニュアル]バナーをクリックし[iSTAR-2セットアップ手引きダウンロード]を参照してください。

Ｑ


「御客様のIDは利用制限がされています。QAセンタまたは最寄の営業所までご連絡ください」と出てログインができません。
Ａ

５回以上パスワードを間違われますと利用制限がかかるようになっております。iS-2QAセンタまでご連絡ください。（iS-2QAセンタ フリーダイヤル： 0120-56-2920）

iSTAR-2初期設定について

Ｑ
iS-2の初期設定画面で、お客様コード・ユーザーコード・パスワードを入力し[OK]をクリックすると、「利用者ＣＤもしくはパスワードに誤りがあります」や、「入力された内容は間違っているか、サーバで準備ができていません」と表示されたが、どうすればいいですか？
Ａ お客様コード・ユーザーコード・パスワードの入力が間違っています。パスワードは半角英数字の組み合わせのため、アルファベットの大文字、小文字の入力ミスの可能性があります。再度パスワードを確認し、入力しなおしてください。

Ｑ
iS-2の初期設定の登録を行う途中「プリンタが使用できません」と表示されたがどうすればいいですか？
Ａ [初期設定]画面の[荷札発行種別]選択項目で、お使いになるサーマルプリンタの機種が正しく選択されているかご確認ください。また、プリンタのインストールを行った時と同じユーザーアカウントを使い、お使いのパソコンを起動させているかもご確認ください。
上記に該当しない場合、プリンタの再セットアップを行ってください。

お客様コード・ユーザーコード・パスワードについて
Ｑ
お客様コード・ユーザーコード・パスワードとは何ですか？
Ａ

お客様認証を行うために必要な情報です。
初期登録以降も、iS-2ユーザ専用ページにログイン時に必要な情報ですので、サーマルプリンタと一緒にお届けしている別紙[最初にお読みください]の用紙（お客様コード・ユーザーコード・パスワード記載の用紙）は大切に保管してください。尚パスワードはセキュリティ強化の為、半年に１度パスワードの変更を行って頂いております。

パスワード変更について

Ｑ
パスワードの変更はできますか？
Ａ パスワードはiS-2のHome画面の左下[パスワード変更]ボタンからいつでもご変更できます。

Ｑ
iS-2へのログイン時「パスワードの有効期限はXX/XXまでです。パスワードの変更をお願い致します。」と表示されたがどうすればいいですか？
Ａ [OK]ボタンをクリックするとパスワード変更画面が表示されますので、期限までにパスワードの変更を行ってください。（パスワードは、1〜20文字でお客様任意の半角英数字でお決めいただけます）尚、セキュリティー強化のため半年に１度、必ずパスワードの変更を行って頂いております。ご協力お願い致します。

セクション運用について

Ｑ
セクション運用とは何ですか？
Ａ  1台のパソコンで複数のセクションを切り替えながら使用する運用方法です。セクション間で出荷登録を代理で行いたい場合、有効な運用です。

[例]
複数の事務所間で代理出荷を行いたい場合、また事務所と倉庫間で代理出荷を行いたい場合など、セクション運用が有効です。

Ｑ
セクション運用をするにはどうすればいいですか？
Ａ セクション運用には別途契約が必要です。最寄の福山通運営業所ご相談ください。

エントリーについて

Ｑ
エントリー画面に[ラベル印刷]ボタンが表示されていないのですが？[オールプリント][チョイスプリント]が表示されていないのですが？
Ａ iS-2Home画面の左メニュー欄[オプション]をクリック、項目から[端末情報]を選択、[プリンタの有無]で[有]にチェックを付け、ご使用のサーマルプリンタ機種を正しく選択してください。
[プリンタの有無]で[有]の保存ができない場合、プリンタがインストールされていない可能性があります。インストールされていない場合はプリンタのセットアップを行ってください。

Ｑ
100個口以上のラベル（荷札）を発行できますか？
Ａ 1件のデータにつき、99個口までの対応となりますので、100個口以上の場合、データを分割して荷札を発行してください。なお運賃については、最寄の福山通運営業所へお問い合わせください。

クイックエントリーについて

Ｑ
クイックエントリーとは何ですか？
Ａ 同じ出荷登録内容を頻繁にお使いいただく場合、有効な機能です。出荷内容を登録し、デスクトップにある緑色の[iS-2（クイックエントリー）]アイコンからラベル（荷札）発行を行ってください。（通常は赤色の[iS-2]アイコンからラベル（荷札）発行となります）

Ｑ
ライブラリ登録はどこで行うのですか？
Ａ iS-2Home画面の左側メニュー欄[クイックエントリー]をクリックするとクイックエントリー画面が開きます。さらにクイックエントリー画面下の[ライブラリ登録]ボタンをクリックするとライブラリ登録画面が開きますので、[名称]と[登録番号]をお付け頂き、登録を行ってください。

Ｑ
ライブラリ登録は何件までとなっていますか？
Ａ 99件まで登録可能となっています。登録番号が重複した場合、上書きされてしまいますのでご注意ください。

CSVエントリーについて

Ｑ
CSVエントリーとは何ですか？
Ａ 他システムで作成した出荷の内容を、CSVファイルにすることでiS-2に取込み保存することができます。

Ｑ
CSVのエントリフォーマットが欲しいのですが？
Ａ iS-2の操作マニュアルをご参照ください。iS-2Home画面の右上[ヘルプ]ボタンをクリックし、[付録]ボタンをクリックしてください。CSVエントリフォーマットが確認できます。

出荷照会について

Ｑ
出荷照会と何ですか？
Ａ お客様が登録し出荷されたお荷物の状態を確認することができます。

Ｑ
昨日、間違えてラベル（荷札）発行してしまったデータが消せません。荷物を出荷していないので、データは削除したいのですが？
Ａ ラベル発行済みの出荷日を過ぎたデータは、修正、削除をすることができません。最寄の福山通運営業所までお問い合わせください。

Ｑ
2ヶ月前に出荷した荷物のデータがありません。
Ａ 出荷照会は、出荷日から45日間までのデータのみ可能となっております。（出荷した実績一覧表の印刷や出荷実績のCSV出力を定期的に行うことをお勧め致します）

出荷実績印刷・CSV出力について

Ｑ
出荷実績印刷・CSV出力とは何ですか？
Ａ お客様の出荷内容の一覧表として印刷、またCSVデータとしてCSV出力することができます。

Ｑ
出荷実績を印刷したのですが、実績一覧表の画面が全て表示されず一部のみ表示されますがどうしたらいいですか？
Ａ

ラベル（荷札）発行用のサーマルプリンタが[通常使うプリンタ]に設定されている可能性があります。サーマルプリンタが[通常使うプリンタ]になっていれば、普段お使いのプリンタを[通常使うプリンタ]になるよう設定変更してください。

Ｑ
iS-2から出荷実績をCSV出力すると運賃が表示されません。
Ａ iS-2で表示される運賃は概算となっておりますので、運賃のCSV出力はできません。
詳しい運賃については、最寄の福山通運営業所までお問い合わせください。

再発行について

Ｑ
ラベル（荷札）をもう一度発行したいのですが？
Ａ

iS-2Home画面の左側メニュー欄[エントリーデータ]をクリックし、[再発行]を選択すると再発行画面が開きます。再発行したいデータを選び、ラベル（荷札）を再印刷してください。（複数個口のデータで一部のラベル（荷札）が欲しい場合、[個別再発行]ボタンから再印刷してください）。なお、再発行は当日ラベル印刷分のみ可能です。

ラベル（荷札）印刷について

Ｑ
ラベル（荷札）を印刷しても着店コード（ラベル（荷札）左上に印刷される大きな白字の数字）が印字されません、このまま使えますか？
Ａ お客様が登録された郵便番号にお届け先の着店コードが複数存在する場合印字されません。郵便番号、住所に間違いが無ければラベル（荷札）はそのままお使いください。

Ｑ
ラベル（荷札）のバーコードが文字化けしたのですが？
Ａ iS-2の再セットアップを行ってください。

Ｑ
社内のレーザープリンタでラベル（荷札）を発行することはできますか？
Ａ できません。ラベル（荷札）発行専用のサーマルプリンタのみの対応となっております。

Ｑ
ラベル（荷札）を印刷すると用紙全体に黒と白の縦すじが入ってしまうのですが？
Ａ プリンタの電源をOFFにしてください。プリンタ本体の蓋を開け、サーマルヘッド（上蓋内側の緑色のライン）の汚れを付属のクリーニングペンかウェットティッシュ等、アルコールを含んだもので拭き取ってください（※水での拭き取りは行わないでください。故障の原因となります）

Ｑ


SATO製サーマルプリンタで[FEED]ボタンを押して紙送りをすると5ミリくらい手前の[記事]の辺りでカットされてしまうのですが？

Ａ [オフライン]で[FEED]（紙送り）をされた場合、少しズレてカットされるのは問題ありません。[LINE]ボタンを押して[オンライン]に切り替えラベル印刷をしてください。（オンラインでのラベル印刷時に、プリンタがiS-2のページ設定を認識し、正しい位置でカットされる仕組みになっています）

Ｑ
ラベル（荷札）のセット方法は？
Ａ

【SATO製サーマルプリンタの場合（表面が青）】  

  1. ラベル（荷札）は箱とビニール袋から取り出し、プリンタと同じ高さになるよう配置した後、上蓋の半透明の用紙通し（クリアカバー）を開けまっすぐにラベル（荷札）通します。
  2. プリンタ内、左側にある[用紙は必ずこの下を通してください]と書いてあるツメと、右側の小さな黒いツメの下を通します。
  3. プリンタ内、右側にある紫色の用紙ホルダスライドレバーを押しながらホルダ幅を、ラベル（荷札）の用紙幅に調整します。
  4. ラベル（荷札）セット後、プリンタ本体の蓋を閉じ、プリンタの電源を投入します。
  5. ＬＩＮＥボタンを押しＯＮＬＩＮＥランプが消えた状態で[FEED]ボタンを押し、ラベル（荷札）の位置合わせを行ってください。

【Zebra製サーマルプリンタの場合（表面が白）】  

  1. ラベル（荷札）は箱とビニール袋から取り出し、プリンタと同じ高さになるよう配置した後、蓋を開け、本体と蓋の接続部の間からラベル（荷札）を通します。
  2. プリンタ内、プラテンローラ後ろの左右にある用紙ホルダの上にのせ、そのまま手前に引き出し蓋をしっかり閉じ、プリンタの電源を投入します。
  3. セット後は紙送りボタンを押し、ラベル（荷札）位置合わせを行ってください。

テスト印刷について

Ｑ
ラベル（荷札）のテスト印刷はどこからできますか？

お届け先情報について

Ｑ
iS-2のアドレス帳に登録しているデータを確認したいのですが？
Ａ iS-2Home画面の左側メニュー欄[アドレス帳]をクリックし、[お届け先情報]を選択するとお届け先登録画面が開きます。お届け先登録画面の上側[お届け先コード]入力欄の右隣にある[アドレス帳]ボタンをクリックするとアドレス帳画面が開きますので、検索条件を入力するか、検索条件を空欄のままにし[検索]を行ってください。

Ｑ
お届け先情報に登録したデータをCSVで出力したいのですが？
Ａ iS-2Home画面の左側メニュー欄[アドレス帳]をクリックし、[お届け先情報]を選択するとお届け先登録画面が開きます。お届け先登録画面の下側[CSV出力]ボタンをクリックすると、登録しているお届け先のデータ全件をCSVファイルに出力することができます。

Ｑ
お届け先情報を全件削除したいのですが？
Ａ iS-2Home画面の左側メニュー欄[オプション]をクリックし、[エントリーオプション]を選択すると[エントリーオプション]の画面が開きます。[エントリーオプション]画面の[お届け先]項目欄に[全件削除]ボタンがございますので、クリックし削除してください。削除後は復旧できませんので、特別な場合以外は行わないでください。また、事前にCSV出力によるバックアップをお取りください。

お届け先取込（CSV）について

Ｑ
お届け先取込（CSV）とは何ですか？
Ａ 他システムで利用されていたお届け先（荷受人様）の情報、または新規に作成されたお客様情報をCSVファイルに変換し、iS-2に取り込むことができます。

Ｑ
お届け先CSVフォーマットが欲しいのですが？
Ａ iS-2の操作マニュアルをご参照ください。iS-2Home画面の右上[ヘルプ]ボタンをクリック後、[付録]ボタンをクリックしてください。付録１−３ お届け先CSVフォーマットを確認する事ができます。

Ｑ
お届け先のCSV取り込みは何件まで可能ですか？
Ａ

お届け先CSV取込の総件数は、１万件までとなっております。１万件以上件数がある場合、最寄の福山通運営業所までご相談ください。

ご依頼主情報（荷送人情報）について

Ｑ
ご依頼主情報とは何ですか？
Ａ お客様の電話番号、住所、請求先等の情報が、ご依頼主情報となっております。

請求先を振り分けたいたいのですが？
Ａ iS-2Home画面の左側メニュー欄[アドレス帳]をクリックし、[ご依頼主情報]を選択、請求先ごとにご依頼主情報を登録してください。

請求先情報について

Ｑ
請求先情報とは何ですか？
Ａ 福山通運営業所から指定された[得意先コード]を登録します。請求先情報で登録した請求先は[ご依頼主情報]を登録される時にも指定して頂く情報となります。

Ｑ
[得意先コード]には何を入力すればいいですか？
Ａ 必ず福山通運営業所から指定されたコードを入力してください。

Ｑ
[得意先部課コード]には何を入力すればいいですか？
Ａ 福山通運営業所から指定の無い場合は、登録の必要はありません。空欄のまま保存してください。

Ｑ
[得意先部課名]には何を入力すればいいですか？
Ａ

お客様の任意で名前（会社名、部課所名など全角１０文字以内）を付け、ご登録ください。

エントリーオプションについて

Ｑ
エントリーオプションとは何ですか？
Ａ

エントリーオプションでは、下記の各種設定ができます。

  • エントリー画面での登録時のTabキーでのカーソル移動設定。
  • ラベル（荷札）印刷時の印字形式の設定。
  • 帳票印刷の形式、印字形式の設定。
  • 出荷CSV出力形式を設定。
